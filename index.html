FAQ

Perguntas frequentes

O LGPD Sentinel AI envia meus dados para algum servidor?
Não. A IA roda 100% localmente via Ollama + llama3.1. Nenhum dado pessoal ou sensível é enviado para servidores externos. Tudo fica na sua máquina.
Preciso de conhecimento técnico para instalar?
Você precisa ter Docker instalado. Com 3 comandos (git clone, cp .env.example .env, docker compose up) tudo já está rodando. O README tem um guia passo a passo.
A ferramenta substitui um DPO ou advogado?
Não. O LGPD Sentinel AI é uma ferramenta de apoio que automatiza tarefas operacionais de compliance. Para decisões jurídicas complexas, consulte um profissional qualificado.
O projeto é realmente gratuito?
Sim. A versão self-hosted é e sempre será gratuita, licença Apache 2.0. Você pode usar, modificar e distribuir livremente. O plano Pro (cloud gerenciado) é opcional e financia o desenvolvimento contínuo.
Quais artigos da LGPD a ferramenta cobre?
Data Mapping (Art. 5, 7, 11), RIPD/DPIA (Art. 38), Direitos dos Titulares — Art. 18 (acesso, correção, exclusão, portabilidade, oposição e mais). Cobertura está sendo expandida continuamente.
Posso contribuir com o projeto?
Sim! O projeto está aberto a PRs, issues e sugestões. Veja o CONTRIBUTING.md no GitHub para guidelines de contribuição.
